## JAnalytics Web

针对 Web 开发的统计 SDK，集成快速且简易，有效降低开发者的接⼊成本。

## 说明

+ 极光 Web 统计通过接入极光 iframe 方式实现跨域数据共享，iframe display 类型为 none，不会影响您网页的正常显示。
+ JAnalytics Web SDK 所有上报均为即时上报。即触发立马进行上报。
+ 注册成功前产生的事件会等待当前页面注册成功后进行上报。
